# 线性代数 — 行列式（Determinants）知识总结

## 核心直觉：行列式是什么

行列式不是凭空定义的规则，它就是：

> **矩阵作为空间变换时，把体积放大了几倍**

- det = 2 → 体积变成原来2倍
- det = -1 → 体积不变，但方向翻转了
- det = 0 → 空间被压扁（降维）→ 矩阵不可逆

---

## 为什么det=0就不可逆

矩阵奇异 = 空间降维 = 有零行或主元缺失：

```
zero row → 0 = b₃ 这个方程要么无解要么无穷多解
         → Ax = b 没有唯一解
         → 逆矩阵不存在
```

几何上：行向量张不开完整的空间 → 体积=0 → det=0

---

## 3条基本性质（所有其他性质的源头）

| 性质 | 内容 | 直觉 |
|------|------|------|
| 性质1 | det(I) = 1 | 单位变换，体积不变 |
| 性质2 | 交换两行，det变号 | 旋转方向从逆时针变顺时针 |
| 性质3 | 对每一行线性 | 一条边拉伸k倍，体积也k倍 |

---

## 从3条推出来的重要性质

**两行相等 → det = 0**

推导：交换这两行 → 矩阵没变 → 但det要变号 → det = -det → det = 0

几何：两向量平行 → 张不开 → 面积=0

---

**消元不改变行列式**

推导：
```
det[row1 + k·row2] = det[row1] + k·det[row2]
    [row2         ]      [row2]       [row2 ]
                                  ↑
                              两行相同 = 0
```

所以消元 = 加了一个面积为0的部分 = 总面积不变

---

**det(Aᵀ) = det(A)**

行和列对体积的贡献完全对等，所以转置不改变行列式。

---

## 计算方法：消元法

把A消元成上三角U，记录换行次数：

```
det(A) = (-1)^(换行次数) × (所有主元之积)
```

例子：
```
A → 消元 → U = [1   2   3]
                [0  -1  -5]   换了1次行
                [0   0  -2]

det(A) = (-1)¹ × (1)×(-1)×(-2) = -2
```

---

## 乘法性质（重要！）

```
det(AB) = det(A) · det(B)    ← 变换叠加，倍数相乘

det(kA) = kⁿ · det(A)        ← n行都被拉伸，k出现n次
```

注意：det(2A) ≠ 2·det(A)，对3×3矩阵是 2³·det(A) = 8·det(A)

---

## 用行列式推结论的技巧

遇到矩阵方程，**两边同时取行列式**，把矩阵问题变成数字问题：

例1：A² = A
```
det(A²) = det(A)
det(A)² = det(A)
det(A)(det(A)-1) = 0
→ det(A) = 0 或 1
```

例2：AᵀA = I
```
det(AᵀA) = det(I) = 1
det(Aᵀ)·det(A) = 1
det(A)² = 1
→ det(A) = 1 或 -1（不能为0，因为A可逆）
```

满足AᵀA = I的矩阵叫**正交矩阵**，只旋转或翻转，不拉伸。

---

## 与四个子空间的联系

```
rank = r（主元个数）
r < n  →  空间被压缩  →  det = 0  →  不可逆
r = n  →  空间满秩    →  det ≠ 0  →  可逆

列空间维度 = 行空间维度 = rank（永远相等）
```
