# 线性代数笔记 — 二次型与正定矩阵

---

## 一、二次型是什么

### 直觉

线性函数输入向量、输出向量（Ax）。
二次型输入向量、输出**一个数**：

```
f(x) = x^T A x
```

展开后只有平方项和交叉项，没有线性项，没有常数项。

### 具体例子

```
A = [2  1]    x = [x1]
    [1  3]        [x2]

x^T A x = 2x1² + 2x1x2 + 3x2²
```

对角线元素 → 平方项系数
非对角线元素 → 交叉项系数（两个对称位置合并）

### 为什么A必须是对称方阵

- 方阵：维度匹配，x^T (1×n) · A (n×n) · x (n×1) = 一个数
- 对称：非对称矩阵和它的对称化版本给出完全相同的二次型，交叉项自动合并，非对称部分对二次型没有贡献

---

## 二、正定矩阵的定义

对所有非零向量x，x^T A x > 0，则称A是正定矩阵。

**几何直觉：** 不管从哪个方向戳这个矩阵，能量都是正的。

### 反例说明

```
A = [1  3]
    [3  1]
```

对角线全正，但取 x = [1, -1]^T：

x^T A x = 1 - 3 - 3 + 1 = -4 < 0

不是正定的。说明"对角线全正"只是必要条件，不充分。

## 三、三个等价判断条件

```
特征值全正  <=>  主元全正  <=>  顺序主子式全正
```

### 条件1：特征值全正

**推导：**

对称矩阵的特征向量构成标准正交基 q1, q2, ...

任意向量 x = c1·q1 + c2·q2 + ...

第一步，算 Ax：
```
Ax = c1·λ1·q1 + c2·λ2·q2 + ...
```

第二步，算 x^T A x：
```
x^T A x = (c1q1 + c2q2)^T (c1λ1q1 + c2λ2q2)
        = c1²λ1 (q1^T q1) + c1c2λ2 (q1^T q2) + c2c1λ1 (q2^T q1) + c2²λ2 (q2^T q2)
```

因为正交基：qi^T qj = 0（i≠j），qi^T qi = 1，交叉项全消：

```
x^T A x = λ1·c1² + λ2·c2² + ...
```

- 特征值全正 → 每项 >= 0，x≠0时严格 > 0 → 正定
- 有负特征值 λk < 0 → 取 x = qk → x^T A x = λk < 0 → 不正定

### 条件2：主元全正

**核心结论：** 对称矩阵可以分解为 A = L D L^T，其中D的对角线是主元。

**为什么是 L D L^T 而不是 L U：**

A = LU，同时 A = A^T，所以 LU = U^T L^T。

把U拆开：U = D · M（D是对角主元矩阵，M是对角线全1的上三角矩阵）。

由对称性可以推出 M = L^T，所以 U = D L^T，代入得 A = L D L^T。

**推导：**

令 y = L^T x（L可逆，x和y一一对应），代入：

```
x^T A x = x^T L D L^T x = (L^T x)^T D (L^T x) = y^T D y
        = d1·y1² + d2·y2² + ...
```

主元全正 → 正定，逻辑和特征值那条完全一样。

### 条件3：顺序主子式全正

顺序主子式 = 从左上角依次扩大的子矩阵的行列式。

2×2情形：
```
A = [a  b]
    [b  d]

1阶主子式：a
2阶主子式：ad - b²
```

**为什么和主元等价：**

消元后，第k个主元 = 第k阶主子式 / 第k-1阶主子式

所以主元全正 <=> 主子式从上到下保持正（每次相除还是正的）

**例子：**
```
A = [1  3]      1阶：1 > 0  ✓
    [3  1]      2阶：1×1 - 3×3 = -8 < 0  ✗  → 不正定
```

## 四、完整脉络

二次型 → 对称矩阵 → 对角化 → 纯平方项

具体步骤：

A对称，所以 A = Q Λ Q^T（Q是正交矩阵，Λ对角线是特征值）

代入二次型，令 y = Q^T x（换坐标系）：

```
x^T A x = x^T Q Λ Q^T x = y^T Λ y = λ1·y1² + λ2·y2² + ...
```

**几何意思：** Q^T x 是把坐标系换到特征向量方向。换完之后：
- 交叉项全消失
- 只剩纯平方项
- 等高线从斜椭圆变成轴对齐椭圆

### 完整例子

```
A = [2  1]
    [1  2]
```

特征值：det(A - λI) = (2-λ)² - 1 = 0，得 λ1=3，λ2=1

特征向量：
- λ1=3：q1 = [1/√2, 1/√2]^T
- λ2=1：q2 = [1/√2, -1/√2]^T

分解：
```
Q = [1/√2   1/√2]    Λ = [3  0]
    [1/√2  -1/√2]        [0  1]

A = Q Λ Q^T
```

二次型换坐标后：
```
x^T A x = 3y1² + y2²
```

λ1=3>0，λ2=1>0，正定。

## 五、和SVD的连接

A^T A 一定是对称半正定矩阵：

- 对称：(A^T A)^T = A^T A  ✓
- 半正定：x^T (A^T A) x = (Ax)^T (Ax) = ||Ax||² >= 0

这是SVD的出发点。
