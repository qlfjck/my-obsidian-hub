# 线性代数笔记 — 特征值与特征向量

---

## 为什么要研究特征值？

大多数矩阵运算很复杂，但特征值能把矩阵"压缩"成几个数字。

核心动机：A^100 怎么算？直接乘100次太麻烦。但如果 Ax = λx，那：

A^100 · x = λ^100 · x

更强的：如果任意向量 b = c1·x1 + c2·x2，那：

A^100 · b = c1·λ1^100·x1 + c2·λ2^100·x2

矩阵的100次方变成了数字的100次方。**矩阵运算 → 数字运算**。

---

## 什么是特征向量？

被矩阵A作用后，**方向不变**，只是被拉伸的向量。

Ax = λx

x 是特征向量，λ 是特征值。

---

## 怎么找特征值？

### 推导逻辑

Ax = λx 改写成：

(A - λI)x = 0

要有非零解 → (A - λI) 不可逆 → rank < n → det(A - λI) = 0

### 特征方程

det(A - λI) = 0

解这个方程得到所有特征值 λ。

### 怎么找特征向量？

找到 λ 之后，解 (A - λI)x = 0 的零空间，就是对应的特征向量。

---

## 例子

A = [[3, 1], [0, 2]]（上三角矩阵）

A - λI = [[3-λ, 1], [0, 2-λ]]

det = (3-λ)(2-λ) = 0 → λ = 3 或 λ = 2

λ = 3：解 [[0,1],[0,-1]]x = 0 → x2 = 0，x1 自由 → 特征向量 c·[1, 0]

λ = 2：解 [[1,1],[0,0]]x = 0 → x1 = -x2 → 特征向量 c·[1, -1]

**规律：上三角矩阵的特征值就是对角线上的数。**

---

## 两个验算规律

对于 n×n 矩阵：

- **迹（trace）= 特征值之和**：tr(A) = λ1 + λ2 + ... + λn
- **行列式 = 特征值之积**：det(A) = λ1 · λ2 · ... · λn

用途：算完特征值之后快速验算。

---

## 对角化

### 什么是对角化？

把特征向量排成矩阵 S = [x1, x2, ..., xn]，特征值放对角线上组成矩阵 Λ：

AS = SΛ

如果 S 可逆：

**A = SΛS⁻¹**

### 为什么有用？

A^100 = SΛ^100·S⁻¹

Λ^100 就是对角线上每个特征值取100次方，超简单。

### 什么时候能对角化？

S 可逆 ↔ 特征向量线性无关 ↔ 能对角化

**充分条件：n 个特征值互不相同 → 一定能对角化。**

特征值有重复时：λ 重复了 k 次，需要零空间维度至少为 k，否则找不到足够的线性无关特征向量。

### 对角化失败的例子

A = [[1, 1], [0, 1]]，λ = 1（重复两次）

A - I = [[0, 1], [0, 0]]，零空间维度 = 1，只有一个特征向量，S 不可逆，**不能对角化**。

---

## 对称矩阵的特殊性质

对称矩阵（A = Aᵀ）有三个保证：

1. **特征值一定是实数**（不会出现复数）
2. **不同特征值对应的特征向量一定垂直**
3. **一定能对角化**

### 为什么特征向量垂直？

取 λ1 ≠ λ2，对应 x1, x2。

x2·Ax1 = λ1(x2·x1)

利用 A 对称：x2·Ax1 = (Ax2)·x1 = λ2(x2·x1)

所以 (λ1 - λ2)(x1·x2) = 0，λ1 ≠ λ2 → x1·x2 = 0。

### 谱分解

特征向量垂直，单位化后 S 变成正交矩阵 Q（满足 Q⁻¹ = Qᵀ）：

**A = QΛQᵀ**

这是对称矩阵专属的对角化形式。
