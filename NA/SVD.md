# SVD 奇异值分解 — 学习笔记

---

## 一、为什么需要SVD

对称矩阵可以对角化：A = QΛQᵀ，但要求A必须是对称方阵。

**问题：任意矩阵（不对称、不方阵）有没有类似的分解？**

答案是SVD：A = UΣVᵀ

---

## 二、构造思路

任意矩阵A不好直接处理，但AᵀA一定是对称半正定矩阵。

**证明AᵀA是对称半正定：**

- 对称：(AᵀA)ᵀ = Aᵀ(Aᵀ)ᵀ = AᵀA（转置反序法则）
- 半正定：xᵀ(AᵀA)x = (Ax)ᵀ(Ax) = ||Ax||² >= 0（长度平方必然非负）
- "半"而非"正"：当x在A的零空间里时，Ax=0，等号成立

对称矩阵一定能对角化，所以对AᵀA做特征分解：

AᵀA = VΛVᵀ

- V的列：AᵀA的标准正交特征向量（n个，每个n维）
- Λ的对角线：特征值，全部 >= 0（半正定保证）
- 奇异值定义：σᵢ = √λᵢ

---

## 三、关键引理：A保持正交性

**结论：v₁,...,vₙ在Rⁿ里互相垂直，被A映射后Av₁,...,Avₙ在Rᵐ里依然互相垂直。**

**证明：**

(Avᵢ)·(Avⱼ) = vᵢᵀ AᵀA vⱼ

因为vⱼ是AᵀA的特征向量：AᵀAvⱼ = λⱼvⱼ，代入：

= vᵢᵀ · λⱼvⱼ = λⱼ(vᵢᵀvⱼ)

因为V是正交矩阵，i≠j时 vᵢᵀvⱼ = 0，所以：

(Avᵢ)·(Avⱼ) = 0 ✓

**奇异值的几何意义：**

||Avᵢ||² = vᵢᵀ AᵀA vᵢ = λᵢ(vᵢᵀvᵢ) = λᵢ

所以 ||Avᵢ|| = √λᵢ = σᵢ

**σᵢ就是A把单位向量vᵢ拉伸了多少倍。**

---

## 四、SVD公式的推导

定义 uᵢ = Avᵢ/σᵢ（单位化，仅对σᵢ > 0的情况有效）

则 Avᵢ = σᵢuᵢ，写成矩阵形式：

AV = UΣ

两边右乘Vᵀ（V是正交矩阵，V⁻¹ = Vᵀ）：

**A = UΣVᵀ**

---

## 五、各矩阵的形状

A是m×n矩阵，秩为r：

| 矩阵 | 形状 | 内容 |
|------|------|------|
| V | n×n | 列是AᵀA的标准正交特征向量 |
| Σ | m×n | 对角线是奇异值σ₁≥σ₂≥...≥σᵣ>0，其余为0 |
| U | m×m | 列是Rᵐ里的标准正交基 |

**Σ的结构（m=3，n=2，r=2为例）：**
```
Σ = [σ₁  0 ]
    [0   σ₂]
    [0   0 ]
```

Σ是m×n不是方阵，但结构和对角矩阵一样——只有左上角有非零元素。

---

## 六、为什么只有r个非零奇异值

AᵀA的秩 = A的秩 = r

秩为r → AᵀA有n-r维零空间 → 零空间里的向量满足AᵀAvᵢ = 0vᵢ → 对应特征值为0 → 对应奇异值为0

所以：
- σ₁,...,σᵣ > 0（r个非零奇异值）
- σᵣ₊₁,...,σₙ = 0（n-r个零奇异值）

---

## 七、U的完整构造

- u₁,...,uᵣ：通过 uᵢ = Avᵢ/σᵢ 造出，有明确几何意义
- uᵣ₊₁,...,uₘ：从左零空间里补，和u₁,...,uᵣ全部垂直，用Gram-Schmidt凑成标准正交基

第二部分是纯粹为了让U变成方阵而补的，在实际应用里常用"thin SVD"只取前r列。

---

## 八、SVD与四个基本子空间

**这是SVD最深刻的结论：**

```
        A
Rⁿ ---------> Rᵐ

行空间(r维)  ----×σᵢ---->  列空间(r维)
v₁,...,vᵣ                  u₁,...,uᵣ

零空间(n-r维) -----------> 0
vᵣ₊₁,...,vₙ

                             左零空间(m-r维)
                             uᵣ₊₁,...,uₘ
```

| 子空间 | 所在空间 | SVD给出的标准正交基 |
|--------|----------|---------------------|
| 行空间 | Rⁿ | v₁,...,vᵣ |
| 零空间 | Rⁿ | vᵣ₊₁,...,vₙ |
| 列空间 | Rᵐ | u₁,...,uᵣ |
| 左零空间 | Rᵐ | uᵣ₊₁,...,uₘ |

**为什么v₁,...,vᵣ是行空间的基（不是零空间的基）：**

vᵣ₊₁,...,vₙ满足Avᵢ=0，明确在零空间里。V是正交矩阵，所有列互相垂直，所以v₁,...,vᵣ和零空间的基垂直。行空间和零空间互相垂直且合起来撑满Rⁿ，因此v₁,...,vᵣ只能在行空间里。

注意：vᵢ自己在行空间里（Rⁿ），被A映射之后落在列空间里（Rᵐ），两件事同时成立不矛盾。

---

## 九、SVD的展开形式

A = UΣVᵀ 可以展开成：

**A = σ₁u₁v₁ᵀ + σ₂u₂v₂ᵀ + ... + σᵣuᵣvᵣᵀ**

每一项uᵢvᵢᵀ：
- 形状：m×n
- 秩：1（所有列都是uᵢ的倍数，列空间只有uᵢ一个方向）
- 这就是Lecture 3的"列乘行=秩1矩阵"

**任意矩阵 = r个秩1矩阵的加权和，权重是奇异值。**

σ₁ >= σ₂ >= ... >= σᵣ，第一项贡献最大。只保留前k项得到A的最佳k秩近似，是图像压缩、推荐系统的核心原理。

---

## 十、完整脉络回顾

```
消元、秩、四个子空间
        ↓
行列式（体积，可逆性）
        ↓
正交性（投影、最小二乘、Gram-Schmidt）
        ↓
特征值（对角化 A = SΛS⁻¹）
        ↓
对称矩阵（谱分解 A = QΛQᵀ，特征值实数，特征向量垂直）
        ↓
正定矩阵（二次型，三个等价条件）
        ↓
SVD（A = UΣVᵀ，任意矩阵，四个子空间全部搞定）
```

SVD是终点：它把四个基本子空间、投影、特征值统一在一个分解里，对任意矩阵成立。
