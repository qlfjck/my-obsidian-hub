# 国债收益率曲线：基础中的基础

> 国债收益率曲线是对未来不确定性定价的核心工具，定义所有资产估值分母。

### 文章发布时间
2026年2月19日（周四）约03:27 GMT（北京时间约11:27）

## 核心观点
- **国债收益率曲线**是宏观经济的中央处理器，将政策、增长、通胀、财政、风险偏好等信息压缩为收益率与利差，传导至所有风险资产定价。
- 国债收益率本质上是**折现率**，其变化直接决定风险资产的现值（分母），并反映市场对未来不确定性的实时定价。
- 通过观察不同期限国债收益率涨跌及**利差**结构，可精准诊断市场交易主题（政策路径、增长/衰退、通胀、期限溢价/承载能力），远比单纯看股市叙事更具约束力和前瞻性。

### 1. 我们为什么要看国债？
许多人认为国债是**低风险低收益**的产品，只适合保守投资者；也有人知道**国债价格与收益率反向**关系；更有老手提醒债市复杂、变量多，容易亏损。  

作者强调：债市其实是**宏观的中央处理器**，把未来政策、增长、通胀、财政、风险偏好等压缩成收益率曲线与利差，再通过**贴现率**与**融资链条**传导到所有风险资产。  

债券本质是一个**时间工具**，解决跨期错配：有人现在急需钱，有人暂时闲置资金。债券就是把这种时间错配做成可执行、可转让的合同。出借方让渡当前购买力，换取未来更多购买力（本金+利息）。利息既是**购买力让渡的租金**，也是对未来**通胀、违约、流动性**等不确定性的补偿。  

**国债收益率**每天在做的，就是对未来不确定性进行定价，而这种定价成为所有资产再定价的基础。  

主权国债（尤其是美元国债）因国家征税权与法定货币能力，被视为**最接近无违约**资产，衍生四大功能：
- **无风险利率的基础锚定**：定义不同时间线的价格与风险，所有资产估值都要在此曲线上贴现，再加风险溢价。
- **抵押品的骨架**：融资、衍生品交易依赖高质量抵押品，国债是最核心的通用抵押品。
- **储备与流动性资产**：银行、保险、央行外汇储备需要随时变现、低风险资产。
- **货币政策的传导介质**：央行调控短端利率、预期管理、公开市场操作，最终通过国债曲线影响实体融资成本与风险偏好。

国债收益率与风险资产挂钩的核心逻辑：  
风险资产（如股票）本质是获取公司未来现金流或估值的权利。**国债收益率其实是折现率**。  
举例：承诺一年后得100元，
- 折现率=0 → 现值=100元
- 折现率=10% → 现值=$100 / 1.1 \approx 90.9$元

折现率越高，国债越不值钱，风险资产也越不值钱。收益率曲线就是一张**不同期限的折扣表**，负责给时间定价。  

总结为什么要看国债的四大理由：
1. 债市定义了估值的**分母**：风险资产回报 = 无风险利率 + 风险溢价。成长股、久期资产对贴现率变化尤其敏感。
2. 债市决定了**融资条件**：杠杆资金（回购、保证金、风险平价等）受利率水平与波动影响，债市波动上升常导致被迫减仓。
3. 债市经常**先于股市**反映宏观共识变化：债市定价更直接、受约束，不易被叙事影响。
4. 债市能给出**宏观的细分指导**：通过不同期限与利差对比，可直接判断市场在交易衰退、通胀、再通胀、信用、国家信用、增长、融资约束等主题。

### 2. 各期限国债的意义及常见曲线变化
国债收益率是对未来不同时间段风险的定价。出借方希望收益率越高越好（补偿不确定性），借入方希望越低越好（未来购买力不被打折）。  

不同期限代表不同含义：
#### 2.1 3月期（3M）
最接近现金利率，反映当下短期资金供需紧张度 + 近期政策利率预期。  
- 上行 → 现金更贵，短期融资成本更高，资金紧张  
- 下行 → 现金更便宜，短期融资压力缓解  
与增长/走弱无关，主要作流动性验证，或判断3个月内降息/加息预期。

#### 2.2 2年期（2Y）
未来1–2年内政策利率路径的平均值，市场对加息/降息节奏、幅度、持续时间的共识。  
- 上行 → 政策预期紧张，降息推迟或加息重计价  
- 下行 → 政策预期宽松，降息更近/幅度更大  
一句话：**2Y定价央行**。

#### 2.3 10年期（10Y）
最容易看错的期限，混合宏观因素：增长叙事、政府财政结构、市场仓位。  
核心定价对象：
- 真实利率（分母）
- 通胀补偿（购买力）
- 期限溢价（风险）

上行来源不同，含义迥异：
- 来自真实利率 → 估值分母变大，久期资产下跌
- 来自通胀补偿 → 再通胀叙事，但不一定立即杀估值
- 来自期限溢价 → 未来风险补偿抬升，市场修复成本增高

**十年期定价当下宏观预期共识**，但切忌只看涨跌不看来源。

#### 2.4 20年期（20Y）
常被忽略，受发行结构、拍卖、对冲需求、曲线相对价值等技术因素影响大。  
核心定价：中长端供需结构与对冲行为。  
常用于验证国债持仓者的结构性压力（大基金压力），暗示国债投资意愿与发债主权可信度。  
**用来做结构验证**，不宜直接用于通胀等宏观判断。

#### 2.5 30年期（30Y）
定价政府与市场的**长期承诺可靠性**（财政可持续性、长期通胀、制度可信度），以及持有超长期限所需的不确定性补偿。  
同时有长期负债匹配资金（保险、养老金）的结构性买盘。  
- 相对更弱 → 久期风险补偿要求更高，或市场承载能力收缩  
- 相对更强 → 补偿被压低，长端风险被重新接受

#### 2.6 常见四种利差
仅看各期限收益率不足以获取足够信息，需观察**利差**（相对变化），剥离整体平移噪音，定位矛盾，将曲线作为诊断工具。  

重要提醒：利差走阔可能是短端下行（牛陡）或长端上行（熊陡），形状相同含义不同，必须同时看两端谁在动。  

定义（负值=倒挂）：
- `3m10s = 10Y - 3M`
- `2s10s = 10Y - 2Y`
- `10s20s = 20Y - 10Y`
- `10s30s = 30Y - 10Y`

#### 2.6.1 3m10s
当前政策水平与未来预期世界之间的距离，常用作**金融条件与衰退风险的温度计**。  
- 走平/倒挂 → 短端过于紧张，未来路径被压缩（现在紧张、未来不得不放松）  
- 走阔 → 压力缓解，未来路径与风险补偿重新被接受

#### 2.6.2 2s10s
政策预期与中期市场承受力的对比，指向市场能相信央行多久、经济能抗多久。  
- 走平/倒挂 → 政策预期偏紧，承受力被怀疑，常伴随未来降息定价  
- 走阔 → 政策预期转松，或中期利率/补偿上调

#### 2.6.3 10s20s
验证发行结构、拍卖供需、对冲需求、做市承载能力。  
- 走阔 → 20Y相对更弱，供给/拍卖压力、对冲挤压、承载能力收缩  
- 走平/收窄 → 20Y相对更强，结构压力缓和或长久期买盘增强

#### 2.6.4 10s30s
独立出**久期风险**本身。长端单边抬价常预示期限溢价上升、可信度风险。股债双杀时，可能是期限溢价抬升导致分母与补偿同时上涨。  
- 走阔 → 30Y相对更弱，对长期持有补偿要求更高（财政/供给压力、承载能力下降）  
- 走平/收窄 → 长期补偿下压，长端风险被重新接受

### 3. 最常见四种交易主题下的国债收益变化
作者强调判断交易主题需要不断练习，像医生看病一样综合多指标影响。以下是最基础的四套模板，作为入门第一步。

#### 3.1 主题一：政策路径再定价
定价：央行变鹰/鸽？  
主导期限：3M、2Y（短端）  
典型利差：2s10s、3m10s（驱动端在短端）  
常见形态：
- 短端上行更快 → 3M/2Y明显上行，10Y跟随但幅度小，曲线趋平，2s10s收窄
- 短端下行更快 → 2Y先下，10Y未必同步，曲线趋陡，2s10s走阔
- 3M与2Y背离 → 资金松紧与政策预期矛盾  
含义：短端上行→机会成本上升，风险资产需更高补偿；短端下行→分母压力缓解。下一步看10Y跟随来源（真实利率/通胀补偿）及信用同步性。

#### 3.2 主题二：增长/衰退重定价
定价：未来增长下修？经济承受力到顶？  
主导期限：10Y（中端主导），2Y跟随但节奏不同  
典型利差：3m10s、2s10s  
常见形态：
- 衰退/增长下修 → 10Y下行更明显，长端也可能下行，牛平/牛陡
- 增长上修 → 10Y上行更明显，长端跟随，熊平/熊陡
- 2Y不动10Y动 → 政策稳定，宏观共识在中端漂移  
直觉：10Y下行若衰退驱动，信用与盈利可能同时走弱；10Y上行若健康增长，信用未必立即恶化。下一步验证10Y变动来源及信用表现。

#### 3.3 主题三：再通胀/通胀预期重定价
定价：名义利率上行是增长带来的再通胀，还是通胀风险补偿本身上调？  
主导期限：10Y（名义端），向长端扩散  
典型利差：2s10s、10s30s（久期补偿敏感）  
常见形态：
- 名义上行但曲线不一定平 → 10Y上行，2Y未必同步，2s10s走阔
- 长端更差 → 30Y上行>10Y，10s30s走阔，期限补偿/通胀风险上升
- 短端不跟 → 政策路径不变，市场重定价通胀可能性  
直觉：通胀补偿驱动→风险资产分化，盈利可能改善但政策风险上升；长端补偿抬价→分母与久期补偿双升，压估值。下一步看真实利率与信用。

#### 3.4 主题四：期限溢价/承载能力冲击
定价：拿更久不确定性要收多少租金？中介承载能力是否收缩？供给结构是否压迫长端？  
主导期限：30Y（长端主导），10Y被动跟随  
典型利差：10s30s、10s20s  
常见形态：
- 长端显著变弱 → 30Y上行>10Y，10s30s走阔
- 中长端上凸 → 20Y更弱，10s20s走阔，供给压力外溢
- 短端相对稳定 → 2Y可能不同步甚至下行，市场担忧后续政策被迫转鸽  
直觉：**高概率股债双杀**，风险难被经济数据修复，动的是市场结构变量。下一步验证真实利率、美元、信用收紧情况。

### 4. 十年期国债变动的来源判断
10Y最容易让初学者翻车，因为它是合成结果，被短端（政策）、长端（久期风险）、中段（通胀补偿等）多头拉扯。  

作者提到曾尝试拆解久期风险中的通胀、政策、流动性等占比，但认为数学上无意义，仅统计意义。入门篇只提供**三点定位法**，快速识别驱动端。

#### 4.1 三点定位法
把曲线压缩为三个点（2Y、10Y、30Y），看谁动得最大：
- **情形A：短端带动（政策路径主导）**  
  2Y动最大，10Y跟随，30Y跟得少；2s10s收窄，10s30s未必走阔。  
  → 市场重定价降息/加息，10Y被短端拖动。下一步找事件/数据看金融条件变化原因。

- **情形B：长端带动（久期主导）**  
  30Y动最大，10Y跟随，2Y相对稳定；10s30s走阔明显，2s10s变化小。  
  → 对更久不确定性要求更多补偿，10Y被长端拉动。下一步验证是可信度问题还是承载能力收缩。

- **情形C：中段自走**  
  10Y动最显著，2Y与30Y不同幅度跟随；2s10s与10s30s无清晰驱动形状，曲线中间鼓起/塌陷。  
  → 折扣最硬的部分在变化。下一步判断是真实利率上升，还是通胀补偿变化。
