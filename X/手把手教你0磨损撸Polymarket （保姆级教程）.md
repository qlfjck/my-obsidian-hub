### 两种主要刷量方法
- 方法一：**两边挂单法**（风险小，但需实时关注成交）
- 方法二：**吃单冲锋法**（无需盯盘，但可能挂单未成交导致小亏）

### 两边挂单法详细操作（以Huo vs Cha比赛为例）
- 目标：买Huo赢 + 买Cha赢，股份数差距不大，实现**没有损耗**。
- 示例盘口：
  - Huo赢盘口 **57** 赔率处挂 **500股** 买单。
  - Cha赢盘口 **42** 赔率处挂 **500股** 买单。
- 查看未成交订单：右上角 → 现金 → 未成交订单。
- 关键：等待任意一个订单成交后，用另一个号**市价吃进**相同股份的反方向订单对冲。
  - 示例：Huo赢57赔率500股被吃 → 用另一号市价以**43**赔率买500股Cha赢。
  - 计算：57 + 43 = **100** → **0损耗**。
  - 如果两边都成交：57 + 42 = **99** → 赚1个点。
  - 操作不及时（如45赔率才对冲）：57 + 45 = **102** → 亏2个点。
- 建议：搞机器人监控订单成交并实时提醒。

### 吃单冲锋法详细操作（更激进，以BKN vs CLE为例）
- 选择赔率悬殊的市场（如一边**13**，另一边**88**）。
- 操作：
  1. 挂单 **CLE赢 87** 赔率 **500股**（限价挂）。
  2. **市价吃进** **BKN赢 13** 赔率 **500股**（立即成交）。
- 等待CLE赢87赔率500股被吃进实现对冲。
- 优势：赔率悬殊时成本低（如BKN赢13赔率500股成本仅**65u**），即使未完全对冲也能控制损失。
