### 一、基础概念（大白话版）

#### （一）先搞懂上网是怎么回事
打开浏览器输入 `youtube.com`，为什么能看到视频？  
电脑发送请求 → YouTube服务器 → 返回数据 → 电脑。  
通俗：电脑是“寄信人”，YouTube是“收信人”，一来一回就上网了。

#### （二）IP 是什么
IP = 身份证号。每台联网电脑有独一无二的身份证号（如 `180.168.1.2`），标识“我是谁”。  
网站也有身份证号。访问时浏览器先查网站IP，再发请求。

#### （三）IPv4 / IPv6 是什么
表格对比：

- **IPv4**：如 `154.26.181.62`，大多数网站用它
- **IPv6**：如 `2001:0db8:...`，新版号码多，但部分网络不支持

**教程全程用 IPv4**，不用纠结IPv6。

#### （四）VPS 是什么（重点！）
VPS = 在云端租一台**永远开机的电脑**。  
本地电脑关机，VPS还在跑；可在任何地方登录（公司、网吧、手机）。  
全权控制，想装什么软件都行。  
通俗：云端养一台24小时待命的“服务器”，专门帮处理网络请求。

#### （五）静态 IP 是什么
**静态 IP** = 固定不变的身份证号  
**动态 IP** = 每次随机分配  

为什么要静态IP？  
- 从不同地方登录VPS管理需固定IP  
- 很多网站检测IP稳定性，静态更不易被封

#### （六）机场是什么
机场 vs 自建VPS对比表格：

- **机场**：商家搭建，花钱买；节点多；IP多人共用易“变脏”；高峰期可能卡；价格10-100元/月；商家跑路就凉
- **自建 VPS**：自己搭建；只有一个节点（VPS所在地）；专属IP只有你用；速度取决于线路；价格约10-100元/月；VPS不跑路就稳定

通俗：机场=坐公交（人多挤、不稳定）；自建=自己开车（稳定）。

#### （七）本文搭建的是什么（是不是 VPN）
官方：**VLESS + REALITY**协议（加密通信方式）  
通俗：自己搭的“**科学上网通道**”  

与传统VPN：原理类似（效果一样），但协议不同。  
开软件 → 走VPS通道 → 访问海外  
关软件 → 走本地网络 → 访问国内

#### （八）机房 IP / 住宅 IP
对比表格：

- **机房 IP**：云厂商数据中心分配；速度快、带宽大、便宜；适合大多数人（本教程用）
- **住宅 IP**：家庭宽带分配；像“真人”，不易被封；适合需高匿名性的人

**教程用机房IP**，普通人足够，性价比最高。

#### （三）Hostname 报错处理
报错：Invalid Linux Hostname  
原因：只能用字母数字和`-`，无空格/中文/特殊符号  
示例：`vps.yourname.com`、`node-01.example.net`

#### （四）下单
选第一个基础套餐即可。

#### （五）拿到 VPS IPv4 和 SSH 登录凭证
1. IPv4：在控制台详情页找 **Primary IPv4**
2. SSH凭证：
   - Root密码：邮件/控制台找，PowerShell：`ssh root@<IPv4>`，输入yes+密码
   - 或SSH私钥.pem：下载，PowerShell：`ssh -i <路径.pem> root@<IPv4>`
   - 修复UNPROTECTED PRIVATE KEY：右键属性→安全→禁用继承→添加当前用户完全控制

#### 3. 容易踩的坑表格
- usage: ssh ... → 命令缺`root@`
- Are you sure... → 输入yes
- UNPROTECTED... → 修复权限
- Permission denied → 权限问题未修好

### 七、Windows 配置 v2rayN

#### （一）下载
GitHub：https://github.com/2dust/v2rayN/releases  
推荐：`windows-64.zip`（普通Intel/AMD电脑，开箱即用）

#### （二）解压
固定目录，如`D:\Apps\v2rayN\`

#### （三）强制管理员运行
右键exe→属性→兼容性→勾选“以管理员身份运行”

#### （四）导入节点
菜单→从剪贴板导入批量URL  
成功：列表出现节点

#### （五）开启系统代理
底部：系统代理 → **自动配置系统代理**  
路由 → **绕过大陆（Whitelist）**（国内直连，海外走VPS）

#### （六）连通性验证
点“当前延迟”：非`-1ms`（如161ms）  
日志：海外`socks -> proxy`，国内`socks -> direct`

#### （七）底部状态栏全解
- 本地端口：`mixed:10808`
- 启用Tun：日常关闭（省资源）；游戏/UWP/奇葩软件时开
- 系统代理：**自动配置系统代理**（红色，日常必备）
- 路由：**V4-绕过大陆(Whitelist)**（365天推荐）

**终极配置**：Tun关闭、系统代理红色、路由Whitelist

**特殊**：用Antigravity时，Tun和全局打开

### 九、容易踩的坑——排雷必看！！！

#### （一）两个代理抢控制权
机场退出清代理 → 重新在v2rayN设自动配置

#### （二）`-1ms` / `none` 原因
- 端口没通（防火墙/安全组）
- REALITY握手失败（时间偏差）
- 客户端无管理员权限
- 公钥私钥未生成

#### （三）iptables命令错地方
只能在VPS SSH里敲，非PowerShell

#### （四）Let's Encrypt失败
80端口不通/占用 → 确保外网可访问80

#### （五）关v2rayN就没网
正常，关掉=断通道；习惯放托盘后台
